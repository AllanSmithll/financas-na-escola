# Como ensinar orçamento pessoal para alunos

Ensinar orçamento pessoal é uma das formas mais práticas de introduzir educação financeira em sala de aula.

E o melhor: não precisa ser complicado.

## O que é orçamento pessoal?

Orçamento pessoal é simplesmente o controle do dinheiro que entra e sai.

Na prática, significa responder:

- quanto eu ganho?
- quanto eu gasto?
- para onde meu dinheiro vai?

## Como ensinar isso na escola?

Comece com exemplos simples.

Peça aos alunos para imaginarem que recebem um salário mínimo.

Depois, proponha:

- listar gastos mensais
- separar necessidades e desejos
- identificar possíveis excessos

## Atividade prática

Uma atividade simples:

- Definir uma renda fictícia
- Criar uma lista de gastos
- Ajustar o orçamento para não ficar negativo

Isso já gera reflexão imediata.

## Por que isso funciona?

Porque conecta com a realidade.

Os alunos passam a entender:

- limites financeiros
- importância de planejamento
- consequências das escolhas

## Conclusão

Ensinar orçamento não é ensinar matemática.

É ensinar tomada de decisão.
